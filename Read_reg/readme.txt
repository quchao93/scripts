read_reg.rb

Run this scripts need specify the parent folder of register execl file.

Such as "ruby read_reg.rb -d C:\mcu-chipmodel\npi-data\Kinetis\K3S_1M_2.4G\ksdk\K32W022S1M2_M0P"

If not specify the parent folder, the scripts search the execl in the current folder.

Notice: Keep only one register execl in the folder.
